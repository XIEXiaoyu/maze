This maze if for fun. :P

In the input file, in the first line, 7 and 6 mean that there are 7 rows and 6 columns in the maze.

In the second line, number 1 means that it is a wall and can not be passed. Number 0 menas that it is road and can be passed and number 2 means that it is the starting point, i.e, the entrance of the maze. 

In the last line, number 3 means that it is the exit of maze.

Requirements: j->up, k->down, l->left, m->right to move the curser.
